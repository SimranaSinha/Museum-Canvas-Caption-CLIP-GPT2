# Milestone 1 – Baseline Style-Conditioned Image Captioning

## Overview

Milestone 1 focuses on building the baseline pipeline for a style-conditioned image captioning system. The goal is to generate captions for images in different styles using a CLIP image encoder and GPT-2 language model.

In this milestone, captions are generated using **prompt-only style conditioning** without injecting CLIP embeddings into GPT-2. This serves as the baseline for later milestones where embedding injection and fine-tuning will be implemented.

---

## Project Objective

The main research question for this project is:

**Can a CLIP + GPT-2 pipeline generate stylistically distinct captions for the same image using style prefix conditioning, and how does caption quality change across prompt-only, embedding injection, and fine-tuning approaches?**

Milestone 1 focuses only on the **prompt-only conditioning baseline**.

---

## Dataset

**Dataset Used:** Flickr30k (HuggingFace dataset)

**Dataset Processing Steps:**

* Loaded Flickr30k dataset
* Selected a subset of ~1000 image–caption pairs
* Removed captions with fewer than 5 words
* Images converted to RGB using PIL
* Dataset cached for faster loading

---

## Model Architecture

The baseline model consists of:

**Image Encoder**

* CLIP ViT-B/32
* Generates 512-dim image embeddings

**Language Model**

* GPT-2 (base model)
* Used for caption generation
* No fine-tuning in Milestone 1

**Style Conditioning**
Instead of modifying the model architecture, style is controlled using **prompt prefixes**.

### Style Prompt Prefixes

| Style            | Prompt Prefix                                      |
| ---------------- | -------------------------------------------------- |
| Art Critic       | "An art critic observes this painting:"            |
| Casual Visitor   | "A museum visitor says about this painting:"       |
| Children’s Guide | "A tour guide explains this painting to children:" |

---

## Caption Generation Settings

Caption generation uses nucleus sampling with the following parameters:

* Top-p = 0.92
* Temperature = 0.85
* Repetition penalty = 1.2
* Max new tokens = 60

---

## Baseline Results

The baseline captions generated by GPT-2 are fluent and stylistically different depending on the prompt. However, captions are **not grounded in image content yet** because CLIP embeddings are not injected into GPT-2 in this milestone.

This motivates the next milestone where **CLIP embeddings will be injected into GPT-2 as a soft prefix**.
